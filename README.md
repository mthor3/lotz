# Lotz

A lottery simulator that plays real-life lottery games over simulated time — Oregon Megabucks, Powerball, and Mega Millions — with faithful rules, rolling jackpots driven by a realistic co-player market model, and Monte Carlo batch statistics.

Kotlin Multiplatform throughout: a shared simulation engine (`core-sim`) and a Compose Multiplatform UI (`composeApp`) targeting Desktop (JVM), Android, iOS, and Web (wasmJs).
