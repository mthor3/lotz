## Full description

Lotz is an educational lottery strategy simulator for Oregon Megabucks, Powerball, and Mega Millions.

Choose a game, set the number of entries, frequency, budget, and stop condition, then simulate what could happen across many drawings. See simulated spend, prizes, net results, rolling jackpots, and repeatable Monte Carlo batches.

Features:

- Simulate Oregon Megabucks, Powerball, and Mega Millions strategies.
- Set a budget, time horizon, or play-until-jackpot scenario.
- Model drawing schedules, prizes, jackpot rolls, and simulated co-player wins.
- Compare a detailed single simulation with Monte Carlo batch results.
- Replay a run with its random seed.

Lotz is for education and exploration only. It does not sell lottery tickets, accept wagers, provide gambling services, or predict winning numbers. Simulation results do not improve the odds of any real lottery drawing.
